# ❤️ Heart For Ya

A simple interactive heart reveal website built with React, TypeScript, and Vite.

## Features

- Terminal-style opening screen
- Interactive reveal animation
- Heart made from text
- Heartbeat effect
- Background music
- Responsive design

## Run Locally

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Built With

- React
- TypeScript
- Vite
- HTML5 Canvas
